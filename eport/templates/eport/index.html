Hello my name is John Juvi Q. De Villa and I am a programmer and Film Editor.

I have been programming for 4 years and have made a number of projects since then. I first learned about Java but was unable to create a working app with it since I was relatively new to programming. After then switched to Python so I can learn about AI and machine learning. I was able to learn the basic fundamentals in building a neural network but due to lack of proper online resources, I ended up stopping midway to learn something easier in my age. I then started learning basic web design to create some of the projects down below. In the process, I learned HTML, Javascript, and CSS. For my backend I used my knowledge and experience in Python. After learning Web design, I switched to mobile development in the android platform. Rather than using Java, I used C# so I can use the .NET and Xamarin Framework so I can easily create and IOS app when the time comes to grow.

I also enjoy making videos and trying VFX editing in my spare time. I use Hitfilm Pr in most of my videos since it's easier to use and less resource-intensive than the Adobe products. Other than Hitfilm, I also use the Mocha plugin to do my 3D tracking and Rotoscoping. From time to time I also use Adobe After Effects for animations and advance VFX editing.
