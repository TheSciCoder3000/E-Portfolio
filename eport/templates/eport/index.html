Hello my name is John Juvi Q. De Villa and I am a programmer and Film Editor.

I have been programming for 4 years and have made a number of projects since then. I first learned about Java but was unable to create a working app with it since I was relatively new to programming. After then switched to Python so I can learn about AI and machine learning. I was able to learn the basic fundamentals in building a neural network but due to lack of proper online resources, I ended up stopping midway to learn something easier in my age. I then started learning basic web design to create some of the projects down below. In the process, I learned HTML, Javascript, and CSS. For my backend I used my knowledge and experience in Python. After learning Web design, I switched to mobile development in the android platform. Rather than using Java, I used C# so I can use the .NET and Xamarin Framework so I can easily create and IOS app when the time comes to grow.

I also enjoy making videos and trying VFX editing in my spare time. I use Hitfilm Pr in most of my videos since it's easier to use and less resource-intensive than the Adobe products. Other than Hitfilm, I also use the Mocha plugin to do my 3D tracking and Rotoscoping. From time to time I also use Adobe After Effects for animations and advance VFX editing.

Online Store

During the pandemic, me and my family had a hard time financially. We decided to buy local goods such as vegetables and seafood every Saturday then sell them off in our subdivision. At first we had to rely on Facebook groups created by the Avida Sta. Catalina Village to post our products. This made it hard and tedious to update and respond to order messages. Because of this, I decided to create a website for us to use for free.

I desgined this website using HTML, CSS and Javascript. For my backend, the server-side, I used Python since it's the language I'm currently used to before starting web development. I also made use of bootstrap to make styling easier since it gives you CSS classes that you can easily use.

I added basic functions for our website such as displaying the products that are available and their prices. I also implemented an account system so the users can log in and order. With this we won't be using messenger to manage orders since Messenger wasn't really built for that.

Awards Night Teaser

Figma Layout & Prototype

When me and my team decided on making a todo list app back in winter break, I went online to find a good UI designer for free. I stumbled into Figma which is a really cool web app and it even has its own desktop app. Not only that but it's also capable in sharing the prototype and layout with other people. With this we were able to create the layout of our app within a week. After finalizing the layout, features and design, I then went ahead and started coding the app in Visual Studio 2019 using a Xamarin Framework.
